Simulados Online
===================

It's a simple platform totally written with Vue 2.x with Feathers.js

----------

Install
-------------
> git clone --recursive ...
> npm install
> cd server && npm install

Needs to run in different instances of terminal

Client (root of project)
> npm run dev

Server 
> npm start
